# WANISTREAM Client

Frontend application for WANISTREAM YouTube Streaming Panel.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Open browser:**
   ```
   http://localhost:5173
   ```

## Build for Production

```bash
npm run build
npm run preview  # Preview production build
```
